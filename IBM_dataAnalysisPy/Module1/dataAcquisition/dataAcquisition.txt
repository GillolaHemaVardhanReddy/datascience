Data acquisition in data analysis refers to the process of collecting and gathering raw data 
from various sources for the purpose of analyzing and deriving insights

key components are: 
        1. Identification of Data Sources (from internet)
        2. Data Collection Methods (like web scraping, APIs, and sensors or surveys or interviews)
        3. Data Quality Assessment (ensure data is valid)
        4. Data Storage ( ensure to store acquired data in structured manner)
        5. Data Integration (Combining data from different sources to provide a unified view)
        6. Data Preprocessing (Preparing the data for analysis by transforming, normalizing, and structuring it appropriately.)
